# 📱 hospital-app

医院预约挂号系统用户移动端。

基于 uni-app 开发，支持 H5、小程序、App 多端运行。

---

# 🚀 技术栈

- uni-app
- Vue3
- uView UI
- Axios

---

# ✨ 功能模块

- 用户登录 / 注册
- 首页展示
- 科室列表
- 医生列表
- 医生详情
- 在线预约
- 我的预约
- 消息通知
- 用户中心
